## Installation

```bash
$ yarn install
```

- Fill the _.env_ file with variables

## Running the app

```bash
# development
$ yarn start
# or
$ docker compose up

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
